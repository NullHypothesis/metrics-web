Tor Metrics Portal: Status


The network data collected by the Tor Metrics Project can be used to analyze the Tor network status from a few years ago until an hour ago. There are currently two applications for this data: The ExoneraTor tells you whether some IP address was a Tor relay at a given time, and the Consensus Health summarizes information about the latest network consensus voting process. The Last Log of the metrics portal software execution also falls under this category to evaluate how recent and complete the presented data are.
