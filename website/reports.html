Tor Metrics Portal: Reports


The reports linked on this page are designed to interactively analyze the data gathered in the Tor Metrics Project. These reports are generated on demand, partially with user-specified parameters, using Eclipse BIRT, PostgreSQL, and Java (for details see the Docs section). The following reports are available (only pre-generated PDFs and PNGs, because the BIRT Report Viewer is currently not installed):

  • Network size (static report) (PDF, 43K, generated on December 1, 2009)
  • Network size (with configurable time interval) (PDF, 43K, generated on December 1, 2009)
  • Network size (with monthly stats table) (PDF, 43K, generated on December 1, 2009)
  • Relay versions and platforms (2 pages) (PDF, 145K, generated on December 18, 2009)
  • Bridge churn time plot (PDF, 113K, generated on December 1, 2009)
  • Bridge churn histogram (PDF, 30K, generated on December 2, 2009)
  • Advertised bandwidth (PNG, 20K, generated on December 18, 2009)
  • Exit relays (PNG, 20K, generated on December 18, 2009)
  • Relays by country (PNG, 27K, generated on December 18, 2009)
